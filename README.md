A [tessl](https://tessl.io) tile for creating tessl tiles with docs, rules, and skills.

Registry: https://tessl.io/registry/tessl-labs/tile-creator/

## Install

```sh
tessl i tessl-labs/tile-creator
```

## Contributing

1. Open a PR with your changes
2. Bump the `version` in `tile-creator/tile.json` — CI will fail if you don't
3. Merge — CD publishes the new version to the registry automatically
